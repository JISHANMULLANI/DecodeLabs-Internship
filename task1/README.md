# Project 1: Data Cleaning & Preparation

## Objective
The goal of this project is to clean a raw dataset by handling missing values, removing duplicates, and correcting data formats to make the data ready for analysis.

## Tools Used
- Python
- Pandas
- Jupyter Notebook

## Steps Performed

### 1. Missing Value Handling
- Checked the dataset for null values.
- Filled missing values in the CouponCode column with "No Coupon".

### 2. Duplicate Removal
- Identified duplicate records.
- Removed duplicate rows from the dataset.

### 3. Data Format Correction
- Converted the Date column to proper datetime format.
- Removed extra spaces from text columns such as Product, PaymentMethod, and OrderStatus.

### 4. Data Validation
- Verified that the dataset was cleaned successfully.
- Saved the cleaned dataset as Cleaned_Dataset.xlsx.

## Outcome
The dataset is now clean, consistent, and ready for further analysis and visualization.
