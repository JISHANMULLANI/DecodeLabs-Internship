# Project 4: Data Visualization

## Objective
Create visual representations of data to communicate insights clearly.

## Tools Used
- Python
- Pandas
- Matplotlib

## Visualizations Created

### 1. Bar Chart
Shows total sales by product.

### 2. Pie Chart
Shows distribution of payment methods.

### 3. Line Chart
Shows sales trends over time.

### 4. Box Plot
Identifies outliers in TotalPrice.

## Key Insights

- Product sales vary significantly across categories.
- Certain payment methods are preferred by customers.
- Sales fluctuate over time.
- Outliers exist in TotalPrice and represent unusually large orders.

## Outcome

Visualizations helped transform raw data into meaningful insights that are easy to understand and communicate.
