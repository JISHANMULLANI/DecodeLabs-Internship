# Project 2: Exploratory Data Analysis (EDA)

## Objective
Analyze the dataset to understand patterns, trends, and distributions.

## Tools Used
- Python
- Pandas
- Jupyter Notebook

## Analysis Performed

### 1. Basic Statistics
Calculated:
- Mean
- Median
- Count
- Minimum and Maximum values

### 2. Distribution Analysis
Analyzed:
- Product distribution
- Payment method usage
- Order status frequencies

### 3. Outlier Detection
Used the IQR (Interquartile Range) method to identify unusual values in TotalPrice.

### 4. Key Findings
- Dataset contains 1200 records.
- CouponCode column contains missing values.
- Multiple products and payment methods are present.
- Sales values vary significantly across orders.
- Outliers were identified in TotalPrice.

## Outcome
The analysis helped understand customer behavior, sales patterns, and data distribution, preparing the dataset for deeper analysis and visualization.
